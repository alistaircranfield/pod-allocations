Change log

Pods

Each day, the people still to place appear under To allocate — from an Import, or added with the add-person line. Drag them into pods, or click a person then click where they go. Give out the referral phone the same way: drag the dashed phone token onto a person (or click it, then them); to move it, drag the phone straight off one chip onto another, or click it to take it back. If the phone holder needs supervision, an amber supervisor badge appears — put it on whoever is supervising, the same way. Click a chip's SD / LD / N tag to change their shift. The small A means airway trained. Students and work-experience visitors go on via + observer / student… — they show on the rota but never count in numbers.

The dot beside each date: green all checks pass · amber worth a look · red a unit rule is broken. Click it to read the details. Everything saves automatically — watch for "All changes saved" at the top.

Import

Drop in the week's Work Roster 1 Week Excel export from Optima (instructions on the Import page). Names are matched to the staff list automatically, and each day's bench then shows exactly who's working.

Staff

One row per person: airway, phone, nights, transfer, supernumerary, plus start and end dates. This list is what all the safety checks run on. When someone leaves, set their end date and they disappear from the lists by themselves. Password protected — the password is set the first time anyone opens the page.

Print copy

Saves a read-only, nicely formatted copy of the week as a single file — for printing for the wall, or posting into the Teams chat.

Mistakes

Undo reverses your last change, step by step. The Log page records who changed what and when. Only one person can edit at a time — if someone else has Edit open, the button tells you who, and it frees itself within minutes of them finishing. Every edit session starts from the latest saved version, an older copy can never save over a newer one, and editing closes itself after five minutes untouched.
